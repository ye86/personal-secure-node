# 贡献指南

感谢关注 PSN Node。项目仍处在架构验证阶段，安全边界的正确性优先于功能数量和开发速度。

## 适合当前阶段的贡献

- 威胁模型、协议状态机和失败场景分析；
- 身份、撤销、恢复和授权语义；
- 可重复的测试及模糊测试；
- 数据迁移、导出和兼容方案；
- 文档、术语和用户授权交互改进。

## 提交要求

1. Issue 和提交中不得包含私钥、访问令牌或真实个人数据。
2. 新增协议字段时说明版本兼容、重放和降级风险。
3. 新增权限时同时说明撤销、过期和审计方式。
4. 密码学改动应优先采用成熟、经过分析的协议和库。
5. 功能代码应附带测试；安全敏感代码应包含失败路径测试。
6. 不要把“已加密”等同于“已经安全”，文档需明确假设和边界。

## 开发环境

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install -r requirements.txt
python cli.py
```

项目许可证尚未确定。在贡献较大规模代码前，请先与维护者确认未来的贡献许可和版权安排。
